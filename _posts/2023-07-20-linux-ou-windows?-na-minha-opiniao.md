## Um pouco sobre cada um:

### Windows:

Segundo a [Wikipedia](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwjio9GXu56AAxVRALkGHUeaCt8QmhN6BAgXEAI&url=https%3A%2F%2Fpt.wikipedia.org%2Fwiki%2FMicrosoft_Windows&usg=AOvVaw2E2EGc8jkWhsGfuA7cAumG&opi=89978449) o "Microsoft Windows é uma família de sistemas operacionais desenvolvidos, comercializados e vendidos pela Microsoft", lançado em 1985, atualmente sua versão mais recente é o Windows 11

### Linux:

Segundo a [Wikipedia](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwiS1K2KvJ6AAxWyK7kGHUE2CDAQmhN6BAgSEAI&url=https%3A%2F%2Fpt.wikipedia.org%2Fwiki%2FLinux&usg=AOvVaw0neGFPnN48gowFlbYWsQhk&opi=89978449) o "Linux é um termo popularmente empregado para se referir a sistemas operativos ou sistemas operacionais que utilizam o núcleo Linux. O núcleo foi desenvolvido pelo programador finlandês Linus Torvalds, inspirado no sistema Minix", lançado em 1991, existem diversos sistemas operacionais que usam ele como base, inclusive o Android!

## Critérios:

Eu confesso que não sabia muito bem o que usar como critério, mas pretendia usar algo mais abrangente, e pelo menos um mais pessoal, apesar disso, ambos os casos se aplicam as minhas necessidades de uso, dito isso, podemos seguir com os critérios abaixo:

- Disponibilidade de Software
- Facilidade de uso
- Produtividade

## Avaliação:

### Disponibilidade de Software:

Um assunto complexo, afinal ambas as plataformas possuem diversos softwares exclusivos, mas ainda assim, acredito que o Windows seja merecedor dessa, já que ainda é preciso recorrer a camadas de compatibilidade para rodar apps de Windows no Linux, isso se mostra ainda mais agravante no cenário de jogos, onde o uso dessa camada de compatibilidade é maior, mas apesar disso, boa parte dos software que eu uso são multiplataforma, com exceção o Microsoft Office, um problema que venho enfrentado nos últimos tempos, que ainda pretendo solucionar

### Facilidade de uso:

Nesse critério eu poderia considerar empate técnico, mas para evitar julgamentos, o ponto vai para o Windows também, uma vez que no Linux, a facilidade de uso vai depender de qual a Distribuição Linux que você escolher, e o que você pretende fazer, mas ainda assim a vitória vai para o Windows mesmo

### Produtividade:

Aqui vem o critério pessoal, nesse caso o ponto é para o Linux, para a surpresa de 0 (zero) pessoas claro, o que posso dizer é que já me acostumei a linha de comando para certas tarefas, e isso no Linux é melhor que no Windows, além de me sentir mais livre para poder customizar do meu jeito
